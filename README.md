## 📖 About

**Linear Matrix Transformations** is an interactive educational platform that helps students, developers, engineers, and mathematics enthusiasts visualize how matrices transform vectors and coordinate spaces.

Instead of learning transformations purely through equations, users can instantly see how matrices affect vectors, basis vectors, and entire coordinate systems.

Whether you're studying linear algebra, robotics, computer graphics, machine learning, or engineering mathematics, this tool provides an intuitive way to understand transformations.

## 🧮 Mathematical Background

A linear transformation is represented by:

```math
T(x) = Ax
```

Where:

- **A** = Transformation Matrix
- **x** = Input Vector
- **T(x)** = Transformed Vector

Linear transformations form the foundation of:

- Computer Graphics
- Robotics
- Machine Learning
- Physics Simulations
- Engineering Mathematics
- Computer Vision

---

## 🎮 Example Matrices

### Scaling

```text
[2 0]
[0 2]
```

Scales all vectors by 2.

### Rotation (90°)

```text
[ 0 -1 ]
[ 1  0 ]
```

Rotates vectors counterclockwise.

### Reflection

```text
[ 1  0 ]
[ 0 -1 ]
```

Reflects vectors across the x-axis.

### Shearing

```text
[ 1 1 ]
[ 0 1 ]
```

Skews the coordinate system horizontally.
